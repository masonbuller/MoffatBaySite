<h1>Moffat Bay Lodging Website</h1>

<h2>Description</h2>

<p>Utilized Java along with Eclipse, Java Server Pages, HTML, CSS, Bootstrap, and Tomcat to write, design, and deploy a website that organized lodging reservations for a client. MySQL was utilized to store data and servlets were used to handle HTTP requests. Prototyping, mockups, entity relationship diagrams, continuous integration with GitHub, and SCRUM methodologies were used to assist in developing the project.</p>

<p>Served as the team leader, coordinating team communications and facilitating critical group decisions on the tech stack. Developed the backend using servlets and created form designs using Bootstrap, as well as assisting other members in the development process. This project features different custom error messages that were created to cover all potential edge cases. This site was also created to be functional for desktop, as well as mobile.</p>
